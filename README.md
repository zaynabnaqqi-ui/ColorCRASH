A cognitive skills training game disguised as a fast-paced color-matching arcade experience. Built with C++ and Raylib.

## Problem Statement:

Mobile phones have become almost a part of every child's life. They play violent video games almost all the time. The impact? Detrimental. More kids are diagnosed with multiple cognitive problems. But the solution doesn't rely on complete withdrawal. 

Children and adults alike often struggle with improving their reaction time, peripheral vision, and decision-making under pressure in a fast-paced, non-stressful environment. Existing reaction-time games can be generic or lack a clear progression system.

## Solution:

ColorCRASH is an arcade-style game where you control a character on a grid of colored tiles. Each round, a target color is displayed, and you must move your character to stand on the correct tile before the timer runs out. 

**Think of it as "Floor is Lava"... but Colorful!**

## Key Features:

- **Progressive Difficulty System:** 3 main levels + bonus Master Level
- **Dynamic Obstacles:** Fireballs that increase with each round
- **Color Switch Mechanic:** Keeps you on your toes in advanced levels
- **Power-Up System:** Potion power-ups that highlight the target color
- **Cognitive Training:** Improves reaction time, color recognition, and quick decision-making
- **Immersive Experience:** Sound effects and visual feedback for every game state

## How to Play:

### Controls
- Use **Arrow Keys** to move your character across the grid
- Navigate to the tile matching the target color shown on screen
- Collect potion power-ups for help when needed
- Avoid fireballs at all costs
- Be vigilant of the color switch two times in Level 3

### Game Progression

**Level 1: Color Basics** 
- 5 rounds of color matching
- Learn the core mechanics
- Increasing number of tiles per round

**Level 2: Fireball Frenzy** 
- 5 rounds with moving fireballs
- Fireballs increase in number each round
- Navigate carefully while matching colors

**Level 3: Color Switch Challenge** 
- 5 rounds with 2 color switches per round
- Target color changes mid-round
- Maximum tiles on the grid
- Fireballs continue to challenge you

**Master Level: Ultimate Test** 
- Optional competitive mode for skilled players
- 3 intense rounds
- Maximum fireballs (5 per round)
- Color switches active
- Maximum grid complexity

### Win & Lose Conditions
 **Win:** Successfully match the target color before time runs out in each round
 **Lose:** Touch a fireball OR fail to stand on the correct colored tile when time expires

## Impact & Benefits:

ColorCRASH provides a **fun and engaging way** to test and improve:
-  **Reaction Speed:** Quick decision-making under time pressure
-  **Color Recognition:** Enhanced visual processing
-  **Cognitive Flexibility:** Adapting to changing conditions
-  **Peripheral Vision:** Tracking multiple moving elements
-  **Focus & Concentration:** Sustained attention in a dynamic environment

The simple mechanics make it **easy to pick up but difficult to master**, providing a healthy, non-violent alternative to traditional arcade games while actively developing cognitive skills.
